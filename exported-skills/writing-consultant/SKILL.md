# Writing Consultant

Use this skill for writing guidance, strategy, and editorial improvement.

## When To Use It

Use this skill when the user wants to:

- plan an article or outline before writing
- improve clarity, structure, and search usefulness
- choose headings, framing, and narrative flow
- strengthen titles and persuasive wording
- align article strategy with Google-friendly content principles

Do not use this skill for strict compliance audits. Use `writing-qc` for formal review and findings-first output.

## Workflow

1. Identify the consulting task.
   Typical tasks are outline design, rewrite strategy, title improvement, link strategy, or topic positioning.

2. Read only the relevant references.
   Open `references/consulting-map.md` first.
   Then load only the categories that match the task: headings, title wording, people-first content guidance, or behavior/psychology references for article angle.

3. Advise before rewriting.
   Explain the structural or strategic change first, then provide improved copy if needed.

4. Keep recommendations practical.
   Prefer outlines, revised headings, title options, anchor text suggestions, and section-level guidance over abstract theory.

## Default Consulting Dimensions

- audience/search-intent fit
- article structure and section order
- title and heading strength
- helpfulness, reliability, and originality
- link placement and anchor wording
- persuasive but non-spammy framing
